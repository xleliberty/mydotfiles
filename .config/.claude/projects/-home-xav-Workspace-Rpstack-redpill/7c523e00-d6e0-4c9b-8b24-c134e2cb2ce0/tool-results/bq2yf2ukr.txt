**getDiscriminant() (Lines 56-77):**
```php
protected function getDiscriminant(array $parameters): array
{
    // Extract credentials except excluded keys (default: ['password'])
    $credentials = $parameters['credentials'] ?? null;
    if (\is_array($credentials)) {
        $keysExcluded = \is_array($parameters['cache_keys_excluded'] ?? null)
            ? $parameters['cache_keys_excluded']
            : ['password'];
        foreach ($credentials as $key => $value) {
            if (\in_array($key, $keysExcluded)) {
                continue;
            }
            $discriminant[$key] = $value;
        }
    }

    // Get URI (discriminant_uri > base_uri > main_uri)
    $discriminant_uri = $parameters['discriminant_uri']
        ?? ($parameters['base_uri'] ?? ($parameters['main_uri'] ?? null));
    if (null === $discriminant_uri) {
        throw new ApiConfigException('Unable to find api key parameter...');
    }

    $discriminant['uri'] = $discriminant_uri;
    return $discriminant;
}
```

**Cache Key Computation:**
- Uses `CacheUtils::computeKey()` → MD5 hash of `key_value_key_value...`

```php
    private ApiClientInterface $trinityApiClient;

    public function __construct(ApiClientInterface $trinityApiClient)
    {
        $this->trinityApiClient = $trinityApiClient;
    }

    protected function trinityApiCall(string $method, string $url, array $options = []): array
    {
        return $this->trinityApiClient->call(
            $method,
            $url,
            array_merge(static::$requestOptions, $options)
        )->getData();
    }
}
```

## 8. CACHE KEY COMPUTATION

```php
public static function computeKey($input)
{
    // Example input: ['email' => 'user@example.com', 'uri' => 'https://api.example.com']
    
    // Step 1: Map to key_value pairs
    $datas = array_map(function ($v, $k) {
        return $k.'_'.$v;
    }, $input, array_keys($input));
    // Result: ['email_user@example.com', 'uri_https://api.example.com']

    // Step 2: Join with underscore
    $joined = implode('_', $datas);
    // Result: 'email_user@example.com_uri_https://api.example.com'

    // Step 3: MD5 hash
    return md5($joined);
    // Result: '7f8d9c2a1b4e6f3a9c8d7b6e5f4a3c2d'
}
```

## 9. KEY ENTRY POINTS FOR NEW INTEGRATION

To build a new API client (e.g., ElibertyBookingV2), follow these steps:

### Step 1: Create Token Error Strategy
```php
// src/RedpillBundle/ApiClient/Infra/Error/Token/ElibertyBookingV2TokenError.php
final class ElibertyBookingV2TokenError implements TokenErrorInterface
{
    public function isInvalidToken(ApiResponseErrorInterface $apiResponseError): bool
    {
        // Return true if response indicates auth failure
        return 401 === $apiResponseError->getCode()
            && $apiResponseError->isValueMatch('status', ['UNAUTHORIZED', 'TOKEN_EXPIRED']);
    }
}
```

### Step 2: Wire in services.yml
```yaml
eliberty_booking_v2.api.client.jwt.authenticated:
    class: Eliberty\RedpillBundle\ApiClient\Infra\Client\AuthJwtApiClient
    factory: ['@Eliberty\RedpillBundle\ApiClient\Infra\Client\ParameterizedClientFactory', 'createAuthJwtApiClient']
    arguments:
        $tokenError: '@Eliberty\RedpillBundle\ApiClient\Infra\Error\Token\ElibertyBookingV2TokenError'
        $apiParams:
            base_uri: '%env(ELIBERTY_BOOKING_V2_API_BASE_URL)%'
            login_uri: /auth/login
            credentials:
                api_key: '%env(ELIBERTY_BOOKING_V2_API_KEY)%'
                api_secret: '%env(ELIBERTY_BOOKING_V2_API_SECRET)%'
```
